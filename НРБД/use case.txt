# Use cases 

## Получение информации об одном виде кошек

1. Перейти на страницу полной информации пользователь может по ссылке вида `/cats/CAT_ID`. Ссылками такого вида являются элементы списка главной страницы.

2. На странице подробной информации пользователь может получить: 

    * Фотографию кошки.
    * Название вида.
    * Краткое описание.
    * Численные характеристики: отношение к семье, масштабы линьки, сложность ухода, здоровье, интеллект, отношение к детям, отношение к другим животным, игривость.
    * Некоторая статистика: размеры, вес, длительность жизни, происхождение.
    * История.
    * Подробное описание размеров.
    * Подробное описание характера.
    * Подробное описание особенностей здоровья.
    * Особенности ухода.
    * Описание внешности.
    * Особенности взаимодействия с другими животными и детьми.
